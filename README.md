## Pipeline

```
prepare_data.py ──► train_sft.py (LoRA/QLoRA) ──► train_grpo.py (RLVR: verifiable rewards,
                                                   vLLM colocated rollouts)
                                                        │
results/ tables ◄── evaluate.py (vLLM batch) ◄── scripts/merge_adapter.py
```

## Quickstart

```bash
uv sync
uv run python prepare_data.py
uv run python evaluate.py --preset zero_shot_2b --limit 500   # ~10-min smoke run
uv run python train_sft.py --preset sft_2b
uv run python train_grpo.py --preset grpo_2b_clevr
```

## Layout

- `vqar/` — the library: data loaders, metrics, rewards, judge, inference helpers, presets
- `train_sft.py`, `train_grpo.py`, `evaluate.py`, `prepare_data.py` — entrypoints
- `scripts/` — adapter merging, results→tables, HF Hub push
- `results/` — committed metrics JSONs (git SHA + full config per run)
- `tests/` — metric/reward/data unit tests
